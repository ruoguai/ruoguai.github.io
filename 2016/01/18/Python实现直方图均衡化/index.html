Python实现直方图均衡化

  • Python 图像处理 直方图均衡化


技能的储备从点滴开始，多一点记录或许就多一点希望。


  • 直方图操作有助于图像的增强。
    直方图均衡化就是把直方图中“密集”的地方向“稀疏”的地方平均一下。想象一下，有堆沙子，使劲压，压到“近乎”平整为止。

直方图计算函数histogram

a:输入，input，一般是矩阵类型的图像
bins:int类型的整数或者有序列的标量，如果是个int类型的（默认是10），是直方图中使用小区间的数目，如果是有序列的标量是一个标注了左右区间的。
range:bins的区间，默认缺省
normed:若为true，返回结果是可能的密度函数的值，若为false结果是在每一个区间内的样本数量
weights和density。

histogram的返回值：一个矩阵图像hist和一个单精度变量bin_edges表示区间的边缘
